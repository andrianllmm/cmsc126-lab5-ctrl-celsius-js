# CMSC 126 Lab 5

This project is a simple Student Records Management App for CMSC 126 Lab 5.

## Project Structure

```
/
│
├─ index.html
├─ css/
│   └─ styles.css      // all styling
├─ js/
│   ├─ student.js      // holds student data/models
│   ├─ add.js          // handles adding students
│   ├─ search.js       // handles searching students
│   ├─ time.js         // handles date/time utilities
│   └─ main.js         // imports other JS files and initializes the app
└─ assets/
    └─ images/         // optional, for any images/logos
```
